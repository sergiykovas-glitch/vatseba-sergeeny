# Vatseba Sergeeny — лендінг (відео-фон + посилання)

Простий статичний сайт-візитівка: відео-фон (вініл/голка) і кнопки на стрімінги.

## ✏️ Як міняти текст і посилання (без коду)

Відкрий файл **`config.js`** у звичайному блокноті/редакторі та зміни текст **у лапках**:

- `artist` — велике ім'я зверху.
- `tagline` — короткий напис під ім'ям (постав `""` щоб прибрати, або зміни на іншу назву треку).
- `links` — головні кнопки (Spotify, Apple, YouTube...).
- `more` — кнопки під «Show more».
- `socials` — соцмережі.

Кожне посилання: `label` (текст), `url` (саме посилання), `icon` (назва файлу з `assets/icons` без `.svg`), `color` (колір світіння).
Збережи файл — і онови сторінку. Усе.

## 🎬 Відео-фон (інтро → кросфейд → цикл)

На вході грає **інтро** (голка опускається), потім плавно переходить у **цикл** (крутиться вічно). Кинь файли в папку `assets`:

**Комп'ютер (16:9):**
- `bg-intro.mp4` — голка опускається (1 раз)
- `bg-loop.mp4` — крутиться (зациклено)

**Телефон (9:16):**
- `bg-intro-mobile.mp4`
- `bg-loop-mobile.mp4`

Щоб кросфейд був непомітний — генеруй обидва кліпи в **однаковому ракурсі/світлі/швидкості**, інтро має **закінчуватись** так само, як цикл **починається** (голка внизу, платівка крутиться).

Якщо мобільних файлів нема — телефон візьме версії для компа (обрізані по центру). Якщо відео взагалі нема — показується заставка `assets/bg-poster.png`.

## 🌐 Як викласти безкоштовно

**Netlify Drop (найпростіше):** зайди на https://app.netlify.com/drop і перетягни **всю папку**. Отримаєш адресу типу `vatseba.netlify.app` (з безкоштовним HTTPS).

Або **Cloudflare Pages**: https://pages.cloudflare.com → Upload assets.

> Свій домен (напр. `vatseba.com`) можна під'єднати пізніше (~10–15$/рік).

## 🔒 Безпека

Це **статичний сайт** — нема сервера чи бази даних, які можна зламати, отже ризик мінімальний. Додатково вже налаштовано:
- усі зовнішні посилання з `rel="noopener noreferrer"` (захист від tabnabbing);
- **Content-Security-Policy** + `_headers` (X-Frame-Options, nosniff тощо) — обмежують, що сайт може завантажувати;
- екранування тексту в коді;
- безкоштовний **HTTPS** від хостингу.

Файл `_headers` спрацьовує автоматично на Netlify / Cloudflare Pages.

## Файли
- `index.html` — каркас головної сторінки
- `404.html` — сторінка помилки (хостинг показує її на неіснуючих адресах): прозоре «404» + платівка, що крутиться
- `styles.css` — дизайн та анімації
- `config.js` — **тут міняєш текст і посилання**
- `script.js` — логіка (відео, кнопки, офлайн-стан)
- `_headers` — security-заголовки для Netlify / Cloudflare Pages (не чіпай)
- `assets/` — іконки, курсори, фавікон, заставка. **Відео ти додаєш сюди сама** (`bg-intro.mp4` тощо)
