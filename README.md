# Vatseba Sergeeny — лендінг

Статичний сайт-візитівка музиканта: відео-фон (вініл) + посилання на стрімінги.

🌐 **Живий сайт:** https://vatseba-sergeeny.pages.dev
📦 **Хостинг:** Cloudflare Pages, автодеплой з GitHub (`sergiykovas-glitch/vatseba-sergeeny`).

---

## ✏️ Як міняти текст і посилання (без коду, без кредитів)

Найпростіше — **прямо на GitHub у браузері**:

1. Відкрий репозиторій → клікни файл **`config.js`**.
2. Натисни **олівець** (Edit).
3. Зміни текст **між лапками** (ім'я, підпис, посилання).
4. Внизу **Commit changes**.
5. Cloudflare **сам опублікує** оновлення за ~хвилину. 🎉

У `config.js`:
- `artist` — велике ім'я зверху.
- `tagline` — короткий підпис (постав `""` щоб прибрати).
- `links` / `more` / `socials` — посилання.
- `icon` має збігатись із назвою файлу в `assets/icons` (без `.svg`). Якщо іконка не показується — назва не збіглася з файлом.

## 🎬 Відео-фон

**Один склеєний кліп на пристрій** (інтро з опусканням голки + спін-цикл в одному файлі):
- `assets/bg-mobile.mp4` — вертикальний 9:16 (телефон)
- `assets/bg-desktop.mp4` — горизонтальний 16:9 (комп)

Інтро грає **один раз**, далі по колу крутиться лише спін — це робить `script.js` перемоткою на `loopStart`.

> ⚠️ **Якщо переробляєш відео:** `loopStart` у `script.js` (моб `2.0833`, комп `2.7083`) — це момент, де закінчується інтро й починається цикл, **прив'язаний до ключового кадру**. Перекодуєш відео — треба оновити ці числа (перевір ключові кадри: `ffprobe -select_streams v -skip_frame nokey -show_entries frame=pts_time ...`).

Якщо відео нема — показується заставка `assets/bg-poster.jpg`.

## 🔒 Безпека
Статичний сайт без сервера/бази. Налаштовано: CSP, security-заголовки (`_headers`), `noopener noreferrer` на посиланнях, екранування тексту, перевірка схем посилань, безкоштовний HTTPS від Cloudflare.

## Файли
- `index.html` — головна сторінка
- `404.html` — сторінка помилки
- `styles.css` — дизайн та анімації
- `config.js` — **тут міняєш текст і посилання**
- `script.js` — логіка (відео-цикл, кнопки, офлайн-стан)
- `_headers` — security-заголовки для Cloudflare Pages
- `assets/` — відео, заставки, обкладинка для шерингу, іконки, курсори, фавікон
- `.gitignore` — що не потрапляє в репозиторій (вихідні відео, чернетки)
